# jhzcode
这个项目主要包含5个文件，其中index.h为索引类，IVFADC_index.h为搜索类，k-means.h实现聚类，load_file.h实现文件的读写，main.cpp为主函数。
运行时只需在代码基础上建立项目，运行即可实现测试集的搜索。（默认索引结构k1=1024, m=8, k2=256）
由于根据训练集建立索引结构时间较长，所以我已经提前将建立好的索引结构保存在文件dataindex1.txt,dataindex2.txt,dataindex3.txt中，其中每类对应参数在主函数中有标明。
主函数中写好了根据训练集建立索引结构的代码，只是将其注释了，如果想运行只需把注释去掉即可。
